    tidal-api-wrapper

    An unofficial API wrapper for Tidal Music. Tidal does not have a documented public API so so all endpoints have been found using Chrome Dev Network Tools. tidal-api-wrapper can be viewed on NPM or Github. It is written in ES6 with ES7 async/await features and is compiled with babel. It utilizes Mocha and Chai for testing and Travis-CI for continuous integration. It also uses Istanbul and Coveralls for code coverage.

    View the code on Github
